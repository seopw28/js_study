# Day 002: 버튼 클릭으로 색상 변경하기

## 📚 오늘 배울 내용

- 화면 중앙에 요소 배치하기 (Flexbox)
- JavaScript로 버튼 클릭 이벤트 처리
- 버튼 배경색 동적 변경
- 배열과 인덱스를 활용한 색상 순환

---

## 1️⃣ HTML 구조

### 기본 구조
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>APP Name</title>
    <link rel="stylesheet" href="../colors.css">
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <button id="myButton" class="center-button">클릭하세요</button>
    <script src="script.js"></script>
</body>
</html>
```

### 핵심 요소 설명

#### 버튼 요소
```html
<button id="myButton" class="center-button"> 클릭 </button>
```

- `<button>`: 클릭 가능한 버튼 요소
- `id="myButton"`: JavaScript에서 이 버튼을 선택하기 위한 고유 ID
- `class="center-button"`: CSS에서 버튼 스타일을 적용하기 위한 클래스
- `" 클릭 "`: 버튼에 표시되는 텍스트 (고정)

**특징:**
- HTML 구조가 매우 간단함 (버튼 하나만 존재)
- JavaScript로 동적으로 배경색이 변경됨

---

## 2️⃣ CSS 스타일

### body 스타일 (화면 중앙 정렬)
```css
body {
    margin: 0;
    padding: 0;
    background-color: var(--bg-gray);
    font-family: 'Calibri', Arial, sans-serif;
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
}
```

**핵심 개념:**
- `display: flex`: body를 flex 컨테이너로 만들어 자식 요소들을 유연하게 배치
- `justify-content: center`: 가로 방향 중앙 정렬
- `align-items: center`: 세로 방향 중앙 정렬
- `min-height: 100vh`: 화면 전체 높이(100vh)만큼 설정하여 화면 전체를 사용

**결과:** body 내부의 모든 요소가 화면 정중앙에 배치됨

### 버튼 스타일
```css
.center-button {
    padding: 20px 40px;
    font-size: 20px;
    font-weight: bold;
    color: var(--white);
    background-color: var(--light-blue);
    border: 2px solid var(--white);
    border-radius: 10px;
    cursor: pointer;
    height: 7vh;
}
```

**핵심 개념:**
- `padding: 20px 40px`: 위아래 20px, 좌우 40px 여백
- `border-radius: 10px`: 모서리를 둥글게 만듦
- `cursor: pointer`: 마우스를 올렸을 때 손가락 모양으로 변경하여 클릭 가능함을 표시
- `background-color: var(--light-blue)`: 초기 배경색 (JavaScript로 동적으로 변경됨)

**특징:**
- 전환 효과 없음: 클릭 시 색상이 즉시 변경됨
- 호버/클릭 애니메이션 없음: 단순하고 빠른 반응

## 3️⃣ JavaScript 기능

### 전체 코드
```javascript
const button = document.getElementById('myButton');
const colors = ['var(--light-blue)', 'var(--dark-blue)'];
let currentIndex = 0;

button.addEventListener('click', function() {
    currentIndex = (currentIndex + 1) % colors.length;
    button.style.backgroundColor = colors[currentIndex];
});
```

### 코드 설명

#### 1. 버튼 요소 가져오기
```javascript
const button = document.getElementById('myButton');
```
- `document`: HTML 문서 전체를 나타내는 객체
- `getElementById('myButton')`: ID가 "myButton"인 요소를 찾아서 반환
- `const`: 변수를 선언하는 키워드 (값을 변경할 수 없음)

#### 2. 색상 배열 정의
```javascript
const colors = ['var(--light-blue)', 'var(--dark-blue)'];
```
- **배열 (Array)**: 여러 값을 하나의 변수에 저장하는 자료구조
- `[]`: 배열을 만드는 문법
- 각 색상은 순서대로 배열에 저장됨
- 인덱스: 배열의 위치 (0부터 시작)
- CSS 변수 사용: `var(--light-blue)`는 colors.css에 정의된 색상 변수를 참조

**배열 구조:**
```
인덱스:  0                   1
값:    'var(--light-blue)'  'var(--dark-blue)'
색상:   밝은 파란색          어두운 파란색
```

#### 3. 현재 인덱스 변수
```javascript
let currentIndex = 0;
```
- `let`: 변수를 선언하는 키워드 (값을 변경할 수 있음)
- `currentIndex`: 현재 어떤 색상을 보여주고 있는지 추적하는 변수
- 초기값: 0 (첫 번째 색상 'var(--light-blue)')

#### 4. 클릭 이벤트 리스너
```javascript
button.addEventListener('click', function() {
    // 이벤트 발생 시 실행할 코드
});
```
- `addEventListener()`: 요소에 이벤트 리스너를 추가하는 메서드
- `'click'`: 클릭 이벤트를 감지
- `function() {...}`: 클릭 시 실행할 함수 (익명 함수)

#### 5. 인덱스 증가 (순환)
```javascript
currentIndex = (currentIndex + 1) % colors.length;
```
- `currentIndex + 1`: 현재 인덱스에 1을 더함
- `%`: 나머지 연산자 (모듈로 연산)
- `colors.length`: 배열의 길이 (2)
- **나머지 연산의 효과:**
  - 0 → 1 → 0 (순환)
  - 배열의 끝에 도달하면 처음으로 돌아감

**동작 예시:**
```
클릭 1: currentIndex = (0 + 1) % 2 = 1 (어두운 파란색)
클릭 2: currentIndex = (1 + 1) % 2 = 0 (밝은 파란색으로 복귀)
클릭 3: currentIndex = (0 + 1) % 2 = 1 (다시 어두운 파란색)
```

#### 6. 배경색 변경
```javascript
button.style.backgroundColor = colors[currentIndex];
```
- `style.backgroundColor`: 요소의 배경색을 직접 설정하는 속성
- `colors[currentIndex]`: 배열에서 현재 인덱스에 해당하는 색상 값 가져오기
- 버튼의 배경색을 배열의 현재 인덱스 값으로 변경
- **즉시 변경**: 전환 효과 없이 바로 색상이 바뀜
- **인라인 스타일**: `style` 속성을 사용하여 HTML 요소에 직접 스타일 적용

## 🔄 동작 원리

### 1. 초기 상태
- 버튼 배경색이 `var(--light-blue)` (밝은 파란색)로 표시됨
- `currentIndex = 0` (첫 번째 색상)
- 버튼 텍스트는 " 클릭 "으로 고정

### 2. 첫 번째 클릭
1. `addEventListener('click')`이 클릭 이벤트 감지
2. `currentIndex = (0 + 1) % 2 = 1` 계산
3. `button.style.backgroundColor = colors[1]` 실행
4. 버튼 배경색이 즉시 `var(--dark-blue)` (어두운 파란색)로 변경 (애니메이션 없음)

### 3. 두 번째 클릭
1. 클릭 이벤트 감지
2. `currentIndex = (1 + 1) % 2 = 0` 계산
3. 버튼 배경색이 즉시 `var(--light-blue)` (밝은 파란색)로 변경

### 4. 계속 클릭
- 두 가지 색상이 계속 번갈아가며 변경됨
- 0 → 1 → 0 → 1 → ... (무한 반복)

## ✅ 핵심 정리

### HTML
- `<button>`: 클릭 가능한 버튼 요소
- `id` 속성: JavaScript에서 요소를 선택하기 위한 고유 식별자

### CSS
- `display: flex` + `justify-content: center` + `align-items: center`: 화면 중앙 정렬
- `cursor: pointer`: 마우스를 올렸을 때 손가락 모양으로 변경
- **전환 효과 없음**: 클릭 시 즉시 반응

### JavaScript
- `getElementById()`: ID로 요소 선택
- **배열**: 여러 값을 저장하는 자료구조
- `addEventListener()`: 이벤트 리스너 추가
- `style.backgroundColor`: 요소의 배경색을 직접 변경
- **나머지 연산 (%)**: 배열 인덱스를 순환시키는 방법 (2개 색상이므로 0과 1만 반복)

## 💡 추가 학습

### 배열 사용법
```javascript
const arr = ['첫 번째', '두 번째', '세 번째'];
console.log(arr[0]);      // '첫 번째' 출력
console.log(arr.length);   // 3 출력 (배열 길이)
```

### 이벤트 리스너 다른 방법
```javascript
// 방법 1: addEventListener (권장)
button.addEventListener('click', function() {
    // 코드
});

// 방법 2: onclick 속성 (HTML에서 직접)
<button onclick="함수이름()">클릭</button>
```

### style 속성으로 스타일 변경하기
```javascript
// 배경색 변경
button.style.backgroundColor = 'var(--light-blue)';

// 다른 스타일도 변경 가능
button.style.color = 'var(--white)';           // 텍스트 색상
button.style.borderRadius = '15px';            // 모서리 둥글게
button.style.fontSize = '24px';                // 글자 크기
button.style.padding = '30px 50px';            // 여백
```

### 전환 효과 추가하기 (선택사항)
```css
/* 부드러운 전환 효과를 원한다면 추가 가능 */
.center-button {
    transition: all 0.3s ease;
}

/* 호버 효과 추가 (선택사항) */
.center-button:hover {
    transform: translateY(-2px);
    box-shadow: 0 6px 12px rgba(0, 0, 0, 0.3);
}
```

- **즉각적인 반응**: 클릭 시 색상이 바로 변경됨
