## 1️⃣ HTML 구조

### 기본 구조
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>APP Name</title>
    <link rel="stylesheet" href="../colors.css">
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <!-- 위쪽 컨테이너 -->
    <div class="top-container">
        <h2> 고려대학교 </h2>
    </div>

    <!-- 아래쪽 컨테이너 -->
    <div class="bottom-container">
        <h2> 연세대학교 </h2>
    </div>

</body>
</html>
```

### 핵심 요소 설명

#### 위쪽 컨테이너
```html
<div class="top-container">
    <h2> 고려대학교 </h2>
</div>
```
- `<div>`: 구역을 나누는 컨테이너 요소
- `class="top-container"`: CSS에서 위쪽 컨테이너 스타일을 적용하기 위한 클래스
- `<h2>`: 제목 태그로 텍스트를 표시

#### 아래쪽 컨테이너
```html
<div class="bottom-container">
    <h2> 연세대학교 </h2>
</div>
```
- `class="bottom-container"`: CSS에서 아래쪽 컨테이너 스타일을 적용하기 위한 클래스
- 위쪽과 아래쪽 컨테이너가 화면을 반반씩 나눔

**특징:**
- HTML 구조가 매우 간단함 (2개의 컨테이너만 존재)
- 각 컨테이너 안에 텍스트가 중앙에 배치됨

## 2️⃣ CSS 스타일

### body 스타일 (화면을 위아래로 나누기)
```css
body {
    margin: 0;
    padding: 0;
    background-color: var(--bg-gray);
    font-family: 'Calibri', Arial, sans-serif;
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}
```

**핵심 개념:**
- `display: flex`: body를 flex 컨테이너로 만들어 자식 요소들을 유연하게 배치
- `flex-direction: column`: 자식 요소들을 세로 방향(위아래)으로 배치
- `min-height: 100vh`: 화면 전체 높이(100vh)만큼 설정하여 화면 전체를 사용

**결과:** body 내부의 요소들이 위아래로 배치됨

### 위쪽 컨테이너 스타일
```css
.top-container {
    flex: 1;
    display: flex;
    justify-content: center;
    align-items: center;
    background-color: rgb(131, 21, 43);
}
```

**핵심 개념:**
- `flex: 1`: 남은 공간을 균등하게 차지 (화면의 절반을 차지)
- `display: flex`: 컨테이너 내부를 flex 컨테이너로 만들어 텍스트 중앙 정렬 가능
- `justify-content: center`: 가로 방향 중앙 정렬
- `align-items: center`: 세로 방향 중앙 정렬
- `background-color: rgb(131, 21, 43)`: 고려대학교 색상 (빨간색)

### 아래쪽 컨테이너 스타일
```css
.bottom-container {
    flex: 1;
    display: flex;
    justify-content: center;
    align-items: center;
    background-color: rgb(29, 63, 167);
}
```

**핵심 개념:**
- `flex: 1`: 위쪽 컨테이너와 동일하게 화면의 절반을 차지
- `background-color: rgb(29, 63, 167)`: 연세대학교 색상 (파란색)
- 위쪽과 동일하게 텍스트를 중앙에 배치

### 텍스트 스타일
```css
.top-container h2,
.bottom-container h2 {
    color: var(--white);
    font-size: 55px;
    font-weight: bold;
    font-family: 'Calibri', Arial, sans-serif;
}
```

**핵심 개념:**
- `color: var(--white)`: 텍스트 색상을 흰색으로 설정
- `font-size: 55px`: 큰 글자 크기로 설정
- `font-weight: bold`: 글자를 굵게 표시
- 두 컨테이너의 h2 태그에 동일한 스타일 적용

## 🔄 동작 원리

### 1. 화면 구조
- 화면이 위아래로 반반 나뉨
- 위쪽: 고려대학교 (빨간색 배경)
- 아래쪽: 연세대학교 (파란색 배경)

### 2. 레이아웃 배치
1. `body`에 `display: flex`와 `flex-direction: column` 적용
2. 두 컨테이너가 `flex: 1`로 설정되어 각각 화면의 50% 차지
3. 각 컨테이너 내부에 `display: flex` 적용하여 텍스트 중앙 정렬

### 3. 텍스트 중앙 정렬
- `justify-content: center`: 가로 방향 중앙
- `align-items: center`: 세로 방향 중앙
- 두 속성을 함께 사용하여 완벽한 중앙 정렬

### CSS

- `flex: 1`: 공간을 균등하게 분배
- `justify-content: center` + `align-items: center`: 완벽한 중앙 정렬

- 현재 사용하지 않음 (정적 레이아웃)

## 💡 추가 학습

### Flexbox 기본 개념
```css
/* 부모 요소 (flex 컨테이너) */
.container {
    display: flex;
    flex-direction: column;  /* 세로 방향 */
    /* 또는 row (가로 방향) */
}

/* 자식 요소 (flex 아이템) */
.item {
    flex: 1;  /* 남은 공간을 균등하게 차지 */
}
```

### 중앙 정렬 방법
```css
/* 방법 1: Flexbox 사용 (권장) */
.container {
    display: flex;
    justify-content: center;  /* 가로 중앙 */
    align-items: center;       /* 세로 중앙 */
}

/* 방법 2: 텍스트 중앙 정렬 */
.text-center {
    text-align: center;
}
```

### 컨테이너 크기 조절
```css
/* 화면의 특정 비율로 나누기 */
.top-container {
    flex: 2;  /* 2배 더 큰 공간 차지 */
}

.bottom-container {
    flex: 1;  /* 1배 공간 차지 */
}
/* 결과: 위쪽이 2/3, 아래쪽이 1/3 */
```

### 배경색 변경하기
```css
/* RGB 값 사용 */
background-color: rgb(131, 21, 43);

/* 색상 변수 사용 */
background-color: var(--light-blue);

/* 16진수 사용 */
background-color: #83152b;
```
